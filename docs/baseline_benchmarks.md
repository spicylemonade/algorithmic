# Baseline Benchmarks (Item 008)

Three baselines were run for 1,000 games each using fixed seed 42:
- Random policy
- Material-capture heuristic policy
- Shallow depth-1 search policy

Recorded metrics per baseline:
- Win rate / draw rate
- Average branching factor
- Average per-move wall-clock latency (ms)

See `results/item_008_baselines.json` for full numeric outputs.
